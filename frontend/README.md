# 蕉幻 (Banana Slides) 前端

這是蕉幻 AI PPT 生成器的前端應用。

## 技術棧

- **框架**: React 18 + TypeScript
- **構建工具**: Vite
- **狀態管理**: Zustand
- **樣式**: TailwindCSS
- **路由**: React Router
- **拖拽**: @dnd-kit
- **圖標**: Lucide React

## 開始開發

### 1. 安裝依賴

```bash
npm install
```

### 2. 配置環境變量

**注意**：現在不再需要配置 `VITE_API_BASE_URL`！

前端使用相對路徑，通過代理自動轉發到後端：
- **開發環境**：通過 Vite proxy 自動轉發到後端
- **生產環境**：通過 nginx proxy 自動轉發到後端服務

**一鍵修改後端端口**：
只需在項目根目錄的 `.env` 文件中修改 `BACKEND_PORT` 環境變量（默認 5000），前端和後端都會自動使用新端口：

```env
BACKEND_PORT=8080  # 修改為 8080 或其他端口
```

這樣無論後端運行在什麼地址（localhost、IP 或域名），前端都能自動適配，無需手動配置。

### 3. 啟動開發服務器

```bash
npm run dev
```

應用將在 http://localhost:3000 啟動

### 4. 構建生產版本

```bash
npm run build
```

## 主要功能

### 1. 首頁 (/)
- 三種創建方式：一句話生成、從大綱生成、從描述生成
- 風格模板選擇和上傳

### 2. 大綱編輯頁 (/project/:id/outline)
- 拖拽排序頁面
- 編輯大綱內容
- 自動生成大綱

### 3. 詳細描述編輯頁 (/project/:id/detail)
- 批量生成頁面描述
- 編輯單頁描述
- 網格展示所有頁面

### 4. 預覽頁 (/project/:id/preview)
- 查看生成的圖片
- 編輯單頁（自然語言修改）
- 導出為 PPTX/PDF

## 開發注意事項

### 狀態管理
- 使用 Zustand 進行全局狀態管理
- 關鍵狀態會同步到 localStorage
- 頁面刷新後自動恢複項目

### 異步任務
- 使用輪詢机制監控長時間任務
- 顯示實時進度
- 完成後自動刷新數據

### 圖片處理
- 所有圖片路徑需通過 `getImageUrl()` 處理
- 支持相對路徑和絕對路徑

### 拖拽功能
- 使用 @dnd-kit 實現
- 支持鍵盤操作
- 樂觀更新 UI

## 與後端集成

確保後端服務運行在配置的端口（默認 5000）：

```bash
cd ../backend
python app.py
```
