## Docker Compose（推薦）

<Steps>
  <Step title="克隆倉庫">
    ```bash
    git clone https://github.com/Anionex/banana-slides
    cd banana-slides
    ```
  </Step>
  <Step title="配置環境變量">
    ```bash
    cp .env.example .env
    ```
    編輯 `.env` 設置 API 密鑰，詳見[配置說明](/zh/configuration)。
  </Step>
  <Step title="啟動服務">
    **使用預構建鏡像（最快）：**
    ```bash
    docker compose -f docker-compose.prod.yml up -d
    ```

    **或從源碼構建：**
    ```bash
    docker compose up -d
    ```
  </Step>
  <Step title="訪問應用">
    - 前端：[http://localhost:3000](http://localhost:3000)
    - 後端 API：[http://localhost:5000](http://localhost:5000)

    <Note>端口默認為 3000/5000（通過 `.env` 配置），可修改 `FRONTEND_PORT` 和 `BACKEND_PORT` 自定義。</Note>
  </Step>
</Steps>

## 從源碼部署

### 環境要求

- Python 3.10+
- [uv](https://github.com/astral-sh/uv) 包管理器
- Node.js 16+ 和 npm
- 有效的 AI 服務 API 密鑰

### 後端

```bash
cd backend
uv sync
uv run alembic upgrade head && uv run python app.py
```

### 前端

```bash
cd frontend
npm install
npm run dev
```

<Note>
  使用可編輯 PPTX 導出功能上傳 PPTX 文件時，需要安裝 [LibreOffice](https://www.libreoffice.org/)。
</Note>
