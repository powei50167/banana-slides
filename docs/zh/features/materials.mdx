## 支持的文件格式

| 格式 | 說明 |
|------|------|
| PDF | 提取文本和嵌入圖片 |
| DOCX / DOC | 解析文件內容和結構 |
| PPTX / PPT | 提取幻燈片內容 |
| XLSX / XLS / CSV | 解析表格數據 |
| Markdown | 讀取格式化文本和圖片链接 |
| TXT | 純文本內容 |

## 素材工作流程

1. 在首頁或詳細編輯器中上傳文件
2. 後端自動解析提取關鍵內容、圖片和圖表信息
3. 提取的內容作為上下文用於生成幻燈片描述和圖片

## 圖片粘貼

可直接在首頁、大綱編輯器或描述卡片中粘貼圖片，系統會立即識別並整合。

## 風格参考

上傳参考圖片或模板引導整個演示文稿的視覺風格，AI 會匹配参考的配色、布局和設計語言。
