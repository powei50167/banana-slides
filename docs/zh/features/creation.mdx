## 從想法開始

最簡單的方式。在首頁輸入主題，Banana Slides 會：

1. 生成帶章節的結構化大綱
2. 為每頁創建詳細描述
3. 根據描述生成幻燈片圖片

## 從大綱開始

如果已有結構思路：

1. 創建項目，進入大綱編輯器
2. 添加、刪除或重排頁面和章節
3. 用自然語言調整大綱（如"加一節市場分析"）
4. 從大綱生成頁面描述

## 從頁面描述開始

需要最大控制力時，在詳細編輯器中直接編寫每頁描述，告訴 AI 渲染什麼內容、布局和風格。

<Tip>
  在描述中包含具體文字內容，可顯著提升生成幻燈片的文字渲染質量。
</Tip>

## 風格定制

- **参考圖片** — 上傳風格参考圖引導視覺風格
- **模板** — 使用文字預設控制外觀
- **逐頁素材** — 為單頁附加特定圖片或圖表
