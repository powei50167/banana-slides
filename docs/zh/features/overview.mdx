## 創作路徑

支持三種方式開始創建演示文稿：

1. **從想法開始** — 輸入主題，AI 自動生成大綱和頁面描述
2. **從大綱開始** — 提供結構，AI 填充細節
3. **從頁面描述開始** — 編寫詳細描述，AI 生成視覺效果

詳見[創建演示文稿](/zh/features/creation)。

## 自然語言編輯

用口頭描述修改任意幻燈片，無需菜單操作。支持整頁重新生成和區域編輯。

詳見[編輯幻燈片](/zh/features/editing)。

## 素材支持

上傳参考文件（PDF、DOCX、MD、TXT）和圖片，系統自動提取關鍵內容豐富幻燈片。

詳見[素材與文件](/zh/features/materials)。

## 導出

導出為可編輯 PPTX 或 PDF，16:9 比例，開箱即用。

詳見[導出選項](/zh/features/export)。
