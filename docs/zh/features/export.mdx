## 格式

| 格式 | 可編輯 | 說明 |
|------|--------|------|
| PPTX | 是 | 文字、圖片和布局可在 PowerPoint 中編輯 |
| PDF | 否 | 像素級精確輸出，可直接演示 |

兩種格式默認 16:9 比例。

## 可編輯 PPTX（Beta）

可編輯導出會從生成的幻燈片圖片中提取文字，重建為原生 PowerPoint 文本元素，保留：

- 字號、顏色和加粗樣式
- 文字定位
- 表格內容

<Note>
  配置 `BAIDU_OCR_API_KEY` 可獲得最佳效果，詳見[配置說明](/zh/configuration#可編輯-pptx-導出)。
</Note>

## 頁面選擇

可選擇特定頁面導出，無需導出整個演示文稿。在幻燈片預覽中使用多選功能選擇要包含的頁面。
