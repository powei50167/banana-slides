## 生成的文字有亂碼或模糊

- 切換到更高分辨率（推薦 2K）。注意：OpenAI 格式可能不支持調高分辨率，建議使用 Gemini 格式。
- 在頁面描述中包含具體要渲染的文字內容。

## 可編輯 PPTX 導出效果不佳

通常是 API 配置問題。確保 `BAIDU_OCR_API_KEY` 設置正確。詳見[配置說明](/zh/configuration#可編輯-pptx-導出)和 [Issue #121](https://github.com/Anionex/banana-slides/issues/121)。

## 免費版 Gemini API 能用嗎？

免費層僅支持文本生成，圖片生成需要付費。

## 出現 503 錯誤或重試錯誤

查看後端日志定位問題：

```bash
docker logs --tail 200 banana-slides-backend
```

通常是模型配置不正確導致。

## 修改 .env 後不生效

- Docker 部署需重啟容器。
- 如果曾在網頁設置頁中配置過参數，會覆蓋 `.env`，點擊"還原默認設置"可恢複。
